# ShareFFS
A file transfer utility to share files between multiple devices on the network.

![image](https://github.com/user-attachments/assets/852750e0-451c-4a7c-a6c0-3fba44e98f85)

## Usage
1. Clone this repo
2. cd into cloned repo
3. Run "node server.js"
4. Open brower and go to "<IP-of-server> or localhost:4000"
5. Enjoy :)

## Pros:
1. It works!!!!

## Cons:
1. Shit design (skill issues).
2. No security at all, just pray that no bad guys are in the network.
